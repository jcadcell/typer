
PROJECT TITLE: Operation Typer
PURPOSE OF PROJECT: To teach typing by announcing what key was pressed. Can also announce common words if the Enter or space key is pressed.
VERSION or DATE: 0.3
HOW TO START THIS PROJECT: Open this file: ~/dev/greenfoot/OperationTyper
USER INSTRUCTIONS: Click Run and type a key or word. Press Enter or space to hear the word. If Operation Typer doesn't know the word, the sound will be "mmp". To stop the project, click Pause.

PROJECT TITLE: Operation Typer
PURPOSE OF PROJECT: To teach typing by announcing what key was pressed. Can also announce common words if the Enter key is pressed twice.
VERSION or DATE: 0.2
HOW TO START THIS PROJECT: Open this file: Projects\Greenfoot\Operation Typer\project.greenfoot
AUTHORS:
USER INSTRUCTIONS: Click Run and type a key or word. Press Enter twice to hear the word. If Operation Typer doesn't know the word, the sound will be "mmp". To stop the project, click Pause.
